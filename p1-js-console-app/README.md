# P1 - JavaScript Console Application

## 📖 Description
This is a Node.js console application developed as part of the Uplift Code Camp. It demonstrates core JavaScript concepts such as arrays, functions, and control flow.

## 🚀 Features
- Processes user input via the terminal
- Demonstrates JavaScript logic and algorithms
- Outputs results directly to the console

## 🛠️ Technologies Used
- Node.js
- JavaScript (ES6)
